\doxysubsection{Detailed Description}
A class that represents a cell in the grid. This class is used to create a cell in the grid. Each cell is a rectangle with a random color. The color of the cell can change with a given probability. The color of the cell can be pdated based on the average color of its neighbors. The cell can be suspended and resumed.
